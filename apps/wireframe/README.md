# Wireframe

Browser-runnable HTML prototype. Workspace member: `@firebase-desk/wireframe`.

- Open `src/index.html` directly in a browser (no build step).
- Edit markup/templates in `src/index.html`.
- Edit behavior in `src/app.js`.
- Edit styling in `src/styles.css`.

During Phase 1 this app will start consuming `@firebase-desk/repo-mocks` and `@firebase-desk/ui` from the workspace; until then sources are plain HTML/CSS/JS with no dependencies.

See [../../docs/project-structure.md](../../docs/project-structure.md) and [../../docs/live-wireframe.md](../../docs/live-wireframe.md).
