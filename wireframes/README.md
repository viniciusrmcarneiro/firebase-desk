# Wireframes

- Open `src/index.html` in a browser.
- Edit markup/templates in `src/index.html`.
- Edit behavior in `src/app.js`.
- Edit styling in `src/styles.css`.

No build step is required for the current prototype.

> Monorepo note: this directory is scheduled to move to `apps/wireframe/` during Phase 1 of the engineering foundation (see [../docs/project-structure.md](../docs/project-structure.md)). Source files stay as plain HTML/CSS/JS; the move only adds a `package.json` so the wireframe can consume `packages/repo-mocks` and `packages/ui` from the workspace.
